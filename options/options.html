Paero Options

Change the sites displayed on the new tab page here.

Help

Five sites can be displayed on the new tab page. Any site that is enabled will show on the new tab page.

Site URL
URL of the website that will be linked on the new tab page
Site Abbreviation
Two letter abbreviation that will represent the site
Enabled
If checked, shows the site on the new tab page. Ignored otherwise.
